# Flask
Match，欢迎加星~~

本项目使用：
  
  1）requests,scrapy爬取智联，前程无忧招聘信息，总200万数据。
  
  2）Web后台为Python Flask，存储为Mysql数据库（已优化）。
  
  3）数据可视化分析使用Highcharts,Echarts。
